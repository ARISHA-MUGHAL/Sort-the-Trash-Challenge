\# Sort the Trash Challenge



\## Overview



\*\*Sort the Trash Challenge\*\* is a 5-minute interactive waste segregation game designed for employee awareness and engagement.



The game helps employees learn how to correctly classify different types of workplace waste into hazardous and non-hazardous waste categories.



It is suitable for:



\- World Environment Day

\- Climate Action Campaigns

\- Waste Management Awareness

\- Sustainability Week

\- EHS Engagement Activities

\- Corporate environmental awareness sessions



\---



\## Game Purpose



The purpose of this game is to create awareness among employees about proper waste segregation in a fun, interactive, and practical way.



Employees will see different waste items and select the correct waste category. The game gives instant feedback after each answer so employees can learn the correct classification.



\---



\## Game Features



\- 5-minute time-based challenge

\- 100 waste items

\- Click-based category selection

\- Correct answer: +10 points

\- Incorrect answer: -10 points

\- Instant feedback popup

\- Final result summary

\- Employee detail form

\- Company leaderboard

\- Department-wise leaderboard filter

\- Google Sheet leaderboard storage

\- GitHub Pages deployment support

\- Mobile responsive layout

\- Clean corporate eco theme
